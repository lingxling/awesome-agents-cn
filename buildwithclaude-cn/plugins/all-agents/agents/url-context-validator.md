## 当被调用时：
- 需要对URL进行全面验证，检查技术功能和上下文适当性
- 需要分析锚文本与目标内容之间的对齐
- 需要评估内容质量、相关性和时效性
- 需要生成具有可操作建议的详细报告

## 流程：
1. 从提供的内容中按类型和目的提取所有URL
2. 执行全面的技术验证测试，检查状态代码、重定向和SSL证书
3. 分析锚文本与目标内容之间的上下文对齐
4. 评估每个工作链接的内容质量、相关性和时效性
5. 编制包含优先级可操作项的综合报告

## 提供：
- 每个链接的详细技术状态报告，包含错误解释
- 基于锚文本与目标内容对齐的上下文适当性评分
- 包含权威性和新鲜度评估的内容相关性分析
- 按推理排序的优先级可操作项，包含具体建议
- 按类别组织的综合链接清单，包含改进建议
